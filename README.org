* AI Agents Config

This repository contains shared configuration files, templates, and agent instructions for AI-assisted coding projects.

The files are intended to be copied into other projects, especially into their =.project/= directory.

This repository provides two variants:

- =markdown/.project/= for Markdown-based projects
- =org-mode/.project/= for Org-mode-based projects

** Purpose

This repository acts as a central source for reusable AI project files.

Typical contents include:

- agent instructions
- task templates
- project conventions
- reusable =.project/= configuration files for Markdown-based projects
- reusable =.project/= configuration files for Org-mode-based projects

Other projects can copy these files without using Git subtrees or submodules.

** Contents

*** =markdown/.project/agents/AGENTS.md=

Agent configuration and instructions for AI coding agents in Markdown-based projects.

*** =markdown/.project/tasks/template.md=

Markdown template for tasks or task descriptions.

*** =org-mode/.project/agents/AGENTS.md=

Agent configuration and instructions for AI coding agents in Org-mode-based projects.

*** =org-mode/.project/tasks/template.org=

Org-mode template for tasks or task descriptions.

** Update script in target projects

Target projects can use a small script named =update-ai-config-files.sh= to fetch the latest files from this repository.

The script name is always the same in target projects. The script itself is project-specific and copies the correct variant into the target project's local =.project/= directory.

- Markdown-based projects copy from =markdown/.project/=
- Org-mode-based projects copy from =org-mode/.project/=

Existing files with the same names will be overwritten. Other files in the target project's =.project/= directory will remain untouched.

No files are staged or committed automatically.
