#+TITLE: AI Agents Config

This repository contains shared configuration files, templates, and agent instructions for AI-assisted coding projects.

The files are intended to be copied into other projects, especially into their =.project/= directory.

* Structure

#+begin_src text
.
├── .project/
│   ├── agents/
│   │   └── AGENTS.md
│   └── tasks/
│       └── template.org
└── README.org
#+end_src

* Purpose

This repository acts as a central source for reusable AI project files.

Typical contents include:

- agent instructions
- task templates
- project conventions
- reusable =.project/= configuration files

Other projects can copy these files without using Git subtrees or submodules.

* Usage in other projects

The contents of this repository can be copied into another project by copying the contents of =.project/= into the target project's local =.project/= directory.

Existing files with the same names will be overwritten. Other files in the target directory will remain untouched.

Example:

#+begin_src bash
rsync -a path/to/ai-agents-config/.project/ path/to/target-project/.project/
#+end_src

* Update script in target projects

Target projects can use a small script to fetch the latest files from this repository.

Example:

#+begin_src bash
#!/usr/bin/env bash

set -e

REPO_ROOT="$(git rev-parse --show-toplevel)"
cd "$REPO_ROOT"

TMP=".project-update-tmp"
TARGET=".project"
REPO_URL="https://github.com/USER/ai-agents-config.git"
BRANCH="main"

rm -rf "$TMP"
mkdir -p "$TARGET"

git clone -q --depth=1 --branch "$BRANCH" "$REPO_URL" "$TMP"

rsync -a "$TMP/.project/" "$TARGET/"

rm -rf "$TMP"
#+end_src

The script only copies files and directories from =.project/= into the target project.

It does not run =git add=, =git commit=, or perform any automatic merge.
